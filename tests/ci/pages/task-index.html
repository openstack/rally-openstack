About Rally

Rally is tool & framework that allows one to write simple plugins and combine them in complex tests scenarios that allows to perform all kinds of testing!

Steps to repeat locally

  1. Fetch rally task from here
  2. Fetch rally plugins from here
  3. Install OpenStack and Rally using this instruction
  4. Unzip plugins and put to .rally/plugins/ directory
  5. Run rally task: $ rally task start task.yaml
